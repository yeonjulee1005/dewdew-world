# `Arc`란?

[Arc Browser](https://arc.net/)는 `Chromium` 기반의 잘 제작된 브라우저입니다.
Chrome의 기본 기능(예: `Chrome 확장 프로그램`, Chrome 북마크 등)을 지원하며, `Arc`의 고유한 기능을 제공합니다.

주요 추가 기능에는 화면 분할, `작업 공간 관리`, 계정 기반 작업 공간 저장 및 공유, Chrome 확장 프로그램 관리, `부스트 모드`, 전체 화면 보기 등이 포함됩니다.

자세한 내용은 [Arc Browser](https://arc.net/)를 참조하세요.

## Windows에서는 사용할 수 없었습니다.

안타깝게도, 현재까지 Arc 브라우저는 `MacOS`와 `Linux`에서만 지원되었습니다.
이는 Windows 사용자가 Arc 브라우저를 사용할 수 없다는 것을 의미합니다.

하지만 오늘, 우리는 흥미로운 소식을 전해드립니다: Arc 브라우저의 Windows 베타 버전이 이제 사용할 수 있습니다.

> [Arc 브라우저 베타 테스터 등록 링크](https://browserco.typeform.com/to/mCUIbqPQ?typeform-source=www.isarconwindowsyet.com)

위의 링크를 따라가면, 베타 테스터로 쉽게 등록할 수 있습니다.
