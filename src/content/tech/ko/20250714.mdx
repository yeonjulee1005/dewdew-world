# `Nuxt4`가 왜 오래 걸렸나요?
`Nuxt 코어 팀`은 지난 2024년 6월에 `Nuxt4`를 출시하기를 목표로 했습니다.<br />
지연은 주로 `Nitro3`의 연기된 출시와 관련이 있었으며,<br />
`다양한 버그` 수정, `Nuxt Icon`의 메인 버전 작업, `Nuxt Ui v3`의 출시와 관련이 있었습니다.

# 이미 `Nuxt4`를 사용할 수 있습니다!
실제로, 이전 포스트에서 언급한 것처럼, `nuxt.config.ts`에 다음 코드를 추가하면, `Nuxt4`를 완벽하게 사용할 수 있습니다.<br />

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  // ...
  future: {
    compatibilityVersion: 4,
  },
  // ...
})
```

이미 경험했습니다!<br />
지난 2024년부터 `Nuxt4`를 `미래 버전`으로 사용하고 있으며, 실제로 프로덕션에서 안정적으로 사용할 수 있었습니다.<br />
또한, 아래 포스트에서 설명한 것처럼,<br />
`Nuxt3`에서 `Nuxt4`로 마이그레이션하는 것이 매우 쉽다는 점에 매우 만족했습니다.

[이전 포스트](/tech/20240618)

# `Nuxt4`에 포함된 개선 사항
`Nuxt4`에는 다음과 같은 개선 사항이 있습니다!<br />

1. 새로운 `/app` 디렉토리 구조가 추가되었습니다.
2. `useAsyncData`와 `useFetch`는 더 나은 캐시 및 처리 방법을 제공합니다.
3. 헤드와 태그 최적화는 더 나은 `SEO` 성능을 제공합니다.
4. `Vite 환경 API`가 소개를 위해 준비되고 있습니다.

원래는 `Nuxt4`에 `Nitro3`를 소개하려고 했지만, `Nuxt4`가 공식적으로 출시된 후, `Nitro3`가 즉시 적용될 것입니다.<br />
이는 `Nitro3`가 `Nitro2`보다 더 안정적이고 진보된 버전이기 때문입니다.

# 그럼, `Nuxt3`는 언제까지 지원될까요?
`Nuxt3`와 예정된 `Nuxt4` 및 `Nuxt5`에 대한 지원은 다음과 같이 진행될 것으로 보입니다!<br />
마이그레이션 프로세스가 더 쉬워지고 있기 때문에, 그렇게 짧은 기간을 설정했다고 합니다!

| Version | Support |
| --- | --- |
| Nuxt3 | 2025년 12월 31일까지 지원 |
| Nuxt4 | 2026년 6월 30일까지 지원 |
| Nuxt5 | 아직 계획 없음 |
