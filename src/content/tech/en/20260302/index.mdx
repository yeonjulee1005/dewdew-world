## Before we start,

In December of last year, I redesigned the portfolio site.<br />
As my personal information increased, I implemented a chat feature that allows visitors to directly ask and answer my information through questions and answers.<br />
[Previous article link](/en/tech/20251211)

I was going to share this after more users used it,<br /> but I found that the content was more interesting than expected, so I wrote this article earlier.

## How much did I spend and how many visitors came in? (AI Agent usage)

I think this part is the most interesting part for you.

Actually, I put LLM models like `GPT`, `Gemini`, `Claude` etc., but the LLM model I actually used was `GPT` mostly.<br />
I think this is because I can easily receive answers without changing the model selection.<br />
Let's take a look at the actual usage together!<br />

![OpenAI API usage and cost](/tech/images/20260302-1.png)

Yes!<br />
I used about `$3` for 3 months.<br />
I used about `10 million` tokens.<br />
The cost per conversation was about `$0.006` (about 8.7 won), and the average monthly cost was about `$0.8`.

### Attack, Defense!!!

In fact, from the very beginning, I designed the Agent to only provide responses based on my own data in order to defend against unexpected requests.<br />
Naturally, there was no risk of prompt injection, though there were people who still tried it, haha.<br />
As you may have seen earlier, if an `injection attack` had actually succeeded, the number of tokens used would have skyrocketed—but you can tell that didn't happen!<br />
(Come at me bro!!!)

![Attack, not attack!!!](/tech/images/20260302-8.png)

If you request the questions that were actually attacked, you can see that it doesn't answer as shown below!
